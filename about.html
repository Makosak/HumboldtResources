About

The project was developed as a friendly collaboration between West Humboldt Park community organizations, a Chicago healthcare group, and a Chicago-born GIS student.

Our goal is to provide easy access to community resource information for community members, by community members. Our approach pulls invaluable community resource data from multiple organizations into a unified resource hub connected to this application.

The code for the web application, building from other open source projects, is likewise open and available for further improvement at Github. The project uses HTML/CSS, Javascript, the Google Maps API, Google Fusion Table API, Google Charts API, jQuery, jQuery address, Bootstrap, Chainbuilder (an open source workflow tool from ASU), and is currently served through Github. We recommend starting with the open Eder Template if you'd like to start your own web map!

The Data

This project began with the carefuly cultivated resource lists from our core partners. Data is merged, shared, and further updated by partners and team volunteers. As new data is added, it will continue to be merged, shared, and further updated for open access. Data can be directly viewed on the map or resource directory, and downloaded here.

The Team

Staff from West Humboldt Park Development Council, Our Lady of Angels, and the Northwestern Memorial Community Services group outlined the need and drove the concept. Marynia Kolak leads the technical development of the project. Student interns who've worked on this include Rachel Weisz (Northwestern University) and Gentry Nissen (Arizona State University).
